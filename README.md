* Project Overview
This project focuses on cleaning and preprocessing the Netflix dataset to prepare it for analysis. The dataset contained missing values, inconsistent formatting, and unstructured duration information.
The data was cleaned and structured using Python and Pandas.

* Data Cleaning Steps
Handled missing values
Removed duplicate records
Standardized text columns
Converted date_added to datetime format
Created year_added and month_added columns
Structured duration into:
1. duration_minutes (Movies)
2. duration_seasons (TV Shows)

* Final Dataset
8794 rows
14 columns
No missing values
